Lesson 1: Caesar Cipher

Definition: The Caesar Cipher is a simple substitution cipher where each letter in the plaintext is shifted by a fixed number of positions.

Example: Suppose we have the ciphertext "EBIIL" and we know it's encrypted using a Caesar Cipher with a shift of 1.

Solution:

  1. Identify the shift value (1 in this case).
  2. Shift each letter in the ciphertext backwards by 1: "EBIIL" becomes "DAAHK", revealing the original plaintext "HELLO".

Lesson 2: Substitution Cipher

Definition: The Substitution Cipher replaces each letter with another letter, number, or symbol based on a fixed mapping.

Example: Given the ciphertext "D1F2" and the mapping A=1, B=2, ..., Z=26, decode the message.

Solution:

  1. Use the mapping to decode each number: D=4, 1=A, F=6, 2=B.
  2. The decoded message is "D1F2" → "A4B6".

Lesson 3: Vigenère Cipher

Definition: The Vigenère Cipher uses a keyword to shift letters in a repeating pattern.

Example: Ciphertext "RIJVS" is encrypted using the keyword "KEY". Decrypt the message.

Solution:

  1. Match the keyword's letters to the ciphertext: R-K, I-E, J-Y, V-K, S-E.
  2. Decrypt each letter by shifting backwards according to the keyword: R-K=Q, I-E=G, J-Y=V, V-K=P, S-E=N.
  3. The decrypted message is "RIJVS" → "QGVPN".

Lesson 4: Transposition Cipher

Definition: The Transposition Cipher rearranges characters in the plaintext according to a certain system.

Example: Given the ciphertext "OLLEH" from reversing "HELLO", recover the original message.

Solution:

  1. Reverse the order of the characters in the ciphertext: "OLLEH" → "HELLO".

Lesson 5: One-Time Pad

Definition: The One-Time Pad uses a random key as long as the plaintext and XORs it to encrypt.

Example: Ciphertext "DPMWM" was encrypted using the key "XMCKL". Decrypt the message.

Solution:

  1. XOR each corresponding character in the ciphertext and key to get the plaintext.
  2. The decrypted message is "DPMWM" → "KHOOR".

Lesson 6: RSA (Rivest-Shamir-Adleman)

Definition: RSA is an asymmetric algorithm that uses a public and private key pair for encryption and decryption.

Example: Encrypt and decrypt a message using RSA.

Solution:

  1. Generate two large prime numbers, p and q.
  2. Compute n = p * q and φ(n) = (p - 1) * (q - 1).
  3. Choose an encryption key e that is coprime to φ(n).
  4. Calculate the decryption key d as the modular inverse of e modulo φ(n).
  5. Encrypt the message m by computing c = m^e mod n.
  6. Decrypt the ciphertext c to recover the message m: m = c^d mod n.

Lesson 7: AES (Advanced Encryption Standard)

Definition: AES is a symmetric block cipher that supports various key lengths for encryption and decryption.

Example: Encrypt and decrypt data using AES.

Solution:

  1. Choose a key of the appropriate length for AES (128, 192, or 256 bits).
  2. Use the AES encryption algorithm to encrypt the plaintext data.
  3. Use the AES decryption algorithm to decrypt the ciphertext data.
